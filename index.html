Click here to install X-Preview

X-Preview allows you to mouse-over thumbnail images on E*Hentai to show a full image preview.

Installation (Firefox)

  1. Make sure you have Tampermonkey or Greasemonkey.
  2. Install the userscript by clicking the big install link above.

Installation (Chrome)

  1. Make sure you have Tampermonkey.
  2. Install the userscript by clicking the big install link above.

Troubleshooting

If something seems to be failing, make sure all of your software is up to date. Then, when something else goes wrong, file an issue request.
